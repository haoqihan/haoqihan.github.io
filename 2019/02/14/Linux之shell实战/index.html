  • /var/log/messages // 系统主日志文件
  • /var/log/secure // 认证，安全

  • access.log //nginx访问日志

Http状态码

  • 1** 信息服务收到请求，需要请求者进一步操作
  • 2** 成功，操作被成功接收并处理
  • 3** 重定向，需要进一步的操作以完成请求
  • 4** 客户端错误
  • 5** 服务器错误
